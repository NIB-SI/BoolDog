.. py:function:: booldog2circuit(model)

   Export a BoolDog Boolean model to Networkx DiGraph

   :param model: A BoolDog object representing a Boolean network.
   :type model: booldog:BoolDogModel

   :returns: **graph** -- A networkx.DiGraph with the same nodes as the input network.
             Boolean rules are represented as additional "logical" nodes
             (and, or, not) and edges.
   :rtype: BooleanDiGraph


.. py:function:: resolve_rule(rule, graph)

   Parse a Boolean rule and replace it with "logical" nodes and edges in the
   graph.

   The rule can be nested with brackets, which are resolved first.

   :param rule: A Boolean rule (e.g. "A & B | C")
   :type rule: str
   :param graph: The graph to which the nodes are added
   :type graph: NetworkxBnet

   :returns: **new_s** -- Replacement string (the final node)
   :rtype: str


.. py:function:: resolve_factor(factor, graph)

   Parse a single factor of a Boolean rule and replace it with "logical" nodes
   and edges in the graph.

   The factor can be a list of nodes connected by '&', '|', and '!' operators.

   :param factor: A Boolean factor (e.g. "A & B | C")
   :type factor: str
   :param graph: The graph to which the nodes are added
   :type graph: NetworkxBnet

   :returns: **s** -- Replacement string (the final node representing this factor)
   :rtype: str
