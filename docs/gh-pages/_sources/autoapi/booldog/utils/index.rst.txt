.. py:function:: parameter_to_array(parameter, graph_keys)

   Parameter argument to numpy array

   :param parameter:
                     if array:
                         make sure length is n
                     if int or float:
                         return an array of length n with value
                     if dict:
                         returns an array of length n with values set according to
                     keys (nodes indexed by graph_keys), and the rest set to 'default' key
   :type parameter: array, int, float or dict
   :param graph_keys:
   :type graph_keys: dict

   :returns: **parameter_array** -- array of length n
   :rtype: numpy array
